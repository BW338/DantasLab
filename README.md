# DANTAS LAB

Sitio web de presentación para DANTAS LAB - Research & Development

## Desarrollo

```bash
npm install
npm run dev
```

## Deploy en Netlify

1. Sube el proyecto a GitHub
2. Conecta el repositorio en Netlify
3. Netlify detectará automáticamente la configuración de `netlify.toml`
4. El sitio se construirá y desplegará automáticamente

## Estructura

- `/src/components` - Componentes React
- `/src/App.jsx` - Componente principal
- `/src/index.css` - Estilos globales
- `netlify.toml` - Configuración de deploy
