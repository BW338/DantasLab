# Sistema de Responsividad - Dantas Lab

## Breakpoints

- **Desktop**: > 1024px (padding: 230px)
- **Tablet**: 768px - 1024px (padding: 60px)
- **Mobile**: < 768px (padding: 30px)

## Variables CSS

```css
--side-padding: 230px (desktop)
--side-padding-tablet: 60px
--side-padding-mobile: 30px
```

## Componentes Actualizados

### Header
- Desktop: Horizontal layout, height 266px
- Tablet/Mobile: Vertical layout, auto height

### Hero
- Desktop: Fixed dimensions
- Tablet/Mobile: Fluid width, adjusted typography

### Research
- Desktop: 3 columns
- Tablet: 2 columns
- Mobile: 1 column stacked

### Quote
- Desktop: Centered, fixed width
- Tablet/Mobile: Full width with padding

### About
- Desktop: Side by side (text + image)
- Tablet/Mobile: Stacked vertically

### Team
- Desktop: Horizontal carousel
- Tablet/Mobile: Vertical scroll

### Footer
- Desktop: 3 columns (logo + nav + info)
- Tablet/Mobile: Stacked vertically

## Notas
- Las guías naranjas se ocultan automáticamente en tablet/mobile
- Los debug borders se mantienen en todos los tamaños
